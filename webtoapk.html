Bungkus website jadi project Android

Isi detail di bawah, klik Convert, lalu unduh file .zip berisi project Android Studio siap-build (WebView + icon + manifest). Alat ini menyiapkan projectnya — build APK final dilakukan di Android Studio karena butuh Android SDK yang tidak bisa jalan di browser.

Nama yang muncul di bawah icon aplikasi.
Format terbalik-domain, huruf kecil, tanpa spasi. Contoh: com.namaanda.namaapp
Format tidak valid. Gunakan huruf kecil, angka, titik. Contoh: com.nama.app
URL harus diawali https:// atau http://
Idealnya persegi, min. 192×192px. Jika kosong, dipakai icon default.
Setelah unduh .zip, bagaimana cara jadi APK?
  1. Ekstrak file .zip yang terunduh ke folder di komputer Anda.
  2. Install Android Studio (gratis, dari developer.android.com) jika belum ada.
  3. Buka Android Studio → Open → pilih folder hasil ekstrak.
  4. Tunggu proses Gradle Sync selesai (butuh koneksi internet, sekali di awal).
  5. Klik menu Build → Build Bundle(s)/APK(s) → Build APK(s).
  6. File app-debug.apk akan muncul di folder app/build/outputs/apk/debug/ — siap diinstal ke HP Android.
